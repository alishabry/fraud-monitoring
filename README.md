Учебный проект представляет собой data-driven приложение для анализа мошеннических транзакций на основе построенного хранилища данных и витрин для аналитики.
Для реализации решения использованы Apache Airflow как инструмент оркестрации, PostgreSQL как хранилище данных и Metabase как инструмент аналитической визуализации.

Цель проекта — продемонстрировать полный цикл работы с данными: от загрузки сырых источников до формирования аналитических представлений и визуализации результатов в дашборде. 

Архитектура приложения построена так, чтобы данные можно было не только хранить, но и системно преобразовывать, обогащать и использовать для принятия решений.
